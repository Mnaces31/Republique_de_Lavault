# 🇫🇷 République de Lavault

Portal institucional fictício desenvolvido para praticar HTML, CSS e design de interfaces modernas.

## 📖 Sobre o projeto

A République de Lavault é um país fictício criado para servir como base para estudos de desenvolvimento web.

O portal apresenta:

- Página inicial institucional
- Menu de navegação com dropdown
- Barra de pesquisa
- Banner principal com imagem de destaque
- Layout responsivo
- Identidade visual própria

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Media Queries

## 🎯 Objetivos

Este projeto foi desenvolvido para praticar:

- Estruturação semântica com HTML
- Criação de layouts responsivos
- Componentização visual
- Menus dropdown
- Hero Sections
- Boas práticas de CSS

## 🌍 Sobre Lavault

A République de Lavault é uma nação fictícia inspirada em países francófonos europeus, desenvolvida no jogo Cities Skylines.

O projeto simula um portal governamental moderno contendo informações institucionais, serviços públicos e conteúdo voltado aos cidadãos.
